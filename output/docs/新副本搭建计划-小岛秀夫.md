# 新副本搭建计划 — 小岛秀夫

> 用新编辑器工具搭建天台副本，全程用编辑器产出 .tres，不改代码。

## 一、副本设定

| 项目 | 内容 |
|------|------|
| 副本名称 | 天台副本（Rooftop） |
| 主题 | 放课后的屋顶，夕阳色调 |
| Stage 数量 | 2（简单验证编辑器流程） |
| Boss | 无 Boss，纯杂兵（精英敌人收尾） |
| 色调 | 橙黄暖色（区别于学校副本的红褐冷色） |
| 地图 | 复用 tilemap_builder 简化地图，改色板为橙调 |

## 二、编辑器分工

| 编辑器 | 产出文件 | 内容 |
|--------|---------|------|
| 对话编辑器 | `resources/dialogue/rooftop_intro.tres` | 开场对话（3条：进入 → 发现敌人 → 目标） |
| 对话编辑器 | `resources/dialogue/rooftop_mid.tres` | 中场对话（2条：增援提示） |
| 对话编辑器 | `resources/dialogue/rooftop_end.tres` | 结束对话（3条：清场 → 夕阳 → 下课） |
| 任务编辑器 | `resources/missions/rooftop_mission.tres` | 2 Stage 任务链，挂对话引用 |
| 技能编辑器 | 检查 `BossAttackData` 参数（不新增，只调参验证） |
| 蓝图编辑器 | `resources/blueprints/rooftop_blueprint.tres` | 简单蓝图：入场 → 对话 → 击杀N → 开门 → 对话 |

## 三、关卡内容概要

### 对话内容

**开场对话 (rooftop_intro, 3条)**
1. 主角: "屋顶……一个人都没有。"
2. 系统: "天台区域 · 清除所有敌人"
3. ???: "……（脚步声从空调机后传来）"

**中场对话 (rooftop_mid, 2条)**
1. 系统: "更多敌人正在接近。"
2. 主角: "来吧。"

**结束对话 (rooftop_end, 3条)**
1. 主角: "结束了。"
2. 系统: "天台区域 · 肃清完成"
3. 主角: "夕阳真好啊。"（看向远方）

### 任务链设计

```
MissionChain: rooftop_clear
├── Stage 1: 踏入天台
│   ├── on_activate_dialogue → rooftop_intro.tres
│   ├── Objective 1: 击杀 8 个敌人 (KILL_COUNT)
│   │   └── on_complete_dialogue → rooftop_mid.tres
│   └── on_complete_dialogue → rooftop_end.tres
└── Stage 2: 清扫残余（自动激活）
    ├── Objective 1: 存活 30 秒 (TIME_SURVIVE)
    │   └── completion_action → complete_stage
    └── on_complete → 结算
```

### 蓝图设计

```
[Trigger: OnPlayerEnter]
    → [Action: PlayDialogue → rooftop_intro.tres]
    → [Action: Wait 2s]
    → [Trigger: OnEnemyKilled count=8]
        → [Action: UnlockDoor → rooftop_exit]
        → [Action: PlayDialogue → rooftop_end.tres]
```

## 四、副本选择界面

游戏启动 → DungeonSelectUI（纯代码已创建，见 scenes/ui/dungeon_select_ui.gd）
- 卡片1: 学校副本（体育馆红褐）
- 卡片2: 天台副本（夕阳橙黄）
- 点击进入 → 加载对应 blueprints .tres + mission .tres

## 六、实施步骤

1. 在 Godot 中启用 Combat Tools 插件
2. 打开对话编辑器 → 创建 3 个 DialogueChain .tres
3. 打开任务编辑器 → 创建 2 Stage 任务链，挂对话引用
4. 打开蓝图编辑器 → 创建入场→击杀→开门流程
5. 在 game_manager 中注册 rooftop dungeon_id
6. 启动游戏 → 副本选择界面 → 选天台 → 进入
