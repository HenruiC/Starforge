## 零、故事之前：设计哲学

> "玩家不会读背景设定。玩家会注意到课桌上写到一半的笔记。"
>
> 这份文档不讲"世界观设定"。它讲三件事：
> 1. 玩家在这个学校里会**看到什么**、**感觉到什么**
> 2. 这些看到的和感觉到的东西，如何连成一个**不需要文字的故事**
> 3. 哪些文字必须在、哪些文字绝对不能有
>
> 最好的故事是玩家自己拼出来的。我们只负责把碎片放对位置。

## 一、故事概要

放学铃响了。你和往常一样收拾书包。但校门没有开。操场上站着穿校服的东西——它们曾经是你的同学。教室里课桌上还摊开着上学期的笔记本，墨迹停在某一页的中间。体育馆深处，哨声还在响。体育老师佐藤幸雄，三年二班的班主任，还在等他的学生来上课。五年前他没有离开那扇门。今天，你可以让他下课了。

**一句话版本：** 一个学生回到废弃的学校，发现所有人都在等下课铃——包括那个吹了五年哨子的体育老师。

## 二、这个学校发生了什么

以下内容**不直接告诉玩家**。它们通过环境物件、文字残片、NPC 行为暗示。玩家可能通关后也不知道全部——这没问题。发现本身就是奖励。

### 2.1 五年前的那一天

那是一个星期一。上午第三节课。没有任何预兆。

不是战争。不是病毒。不是政府实验。是一种无法被分类的"异变"——它在一瞬间降临，改变了学校里的一部分人。被改变的人没有死。他们变成了另一种存在：身体还在，意识破碎成单一的本能。有的在走廊上徘徊，有的站在窗边不动，有的重复着生前最后一个动作。

没有逃出去的人不是因为门被锁了。是因为他们不知道自己应该逃。

### 2.2 环境叙事碎片（按区域分布）

这些不是任务目标。它们在那里，玩家可以看，也可以不看。但如果玩家停下来看——他们会拼出故事。

#### 校门广场（y > 1800）

| 物件 | 位置 | 暗示什么 |
|------|------|---------|
| 歪倒的旗杆，国旗还在顶端 | 广场中央旗台 | 星期一早晨的升旗仪式进行到一半 |
| 公告栏上的日期 | 校门右侧 | 日期停在某个星期一。下面压着下周的课表——没有被取走 |
| 自行车棚 | 校门左侧边缘 | 车还在。锁还在。没有人来骑走过 |
| 散落的书包 | 操场跑道附近 | 几个，不是一堆。像是有人放下书包跑了，没回来拿 |

**叙事逻辑：** 这是玩家看到的第一个区域。信息量要少——两三个细节就够了。玩家在第一场战斗后停下来喘气时，目光会扫到这些东西。

#### 教学楼玄关（y > 1500）

| 物件 | 位置 | 暗示什么 |
|------|------|---------|
| 鞋柜，室外鞋还在里面 | 玄关两侧 | 没有人换鞋回家。每一双鞋都在等主人 |
| 运动会照片 | 鞋柜上方贴板 | 去年的。照片里的人穿着运动服在笑 |
| 失物招领箱 | 玄关角落 | 里面有一把没贴标签的伞、一个笔袋、一副眼镜 |
| 职员表 | 墙上的布告栏 | 佐藤幸雄——三年二班班主任。照片还在。一个普通中年男人的脸 |

**叙事逻辑：** 玄关是"来"和"去"的交界。鞋柜是最有力量的物件——每一个上过学的人都知道换鞋意味着什么。"室外鞋还在里面"这一句话比任何恐怖描写都更让人不安。

#### 走廊（y > 1100）

| 物件 | 位置 | 暗示什么 |
|------|------|---------|
| 三年级公告栏 | 走廊墙面上 | 社团招新海报。日期是五年前的。有人在上面贴了便签——"周三部活，别迟到" |
| 教室门牌 | 每间教室门口 | 三年二班。三年三班。三年四班。门都开着。或者关着。你决定要不要推开 |
| 手抄报墙 | 走廊中段 | 上学期的手抄报。主题是"我的梦想"。纸张边缘已经泛黄卷起 |
| 饮水机 | 走廊尽头 | 水桶是空的。水龙头还滴了一滴水——然后没有了 |

**叙事逻辑：** 走廊是过渡空间。玩家在这里不是停留的——是穿过的。物件设计要"余光可见"：不中断移动，但在视野边缘留下印象。

#### 教室（y > 700）

| 物件 | 位置 | 暗示什么 |
|------|------|---------|
| 黑板，粉笔字写到一半 | 教室前方 | "第 3 課 春——"后面的字断了。粉笔掉在地上 |
| 课桌上的笔记 | 学生课桌 | 上学期的笔记。有人画了涂鸦。有人在角落里写了"好无聊" |
| 没批改完的作业本 | 讲台桌上 | 摞成一叠。红笔搁在旁边。批到第 17 本，第 18 本翻开还没批 |
| 抽屉里的合照 | 某个课桌抽屉 | 社团合照。几个学生挤在一起比耶。照片的边角受潮了 |
| 值日生名牌 | 黑板旁边 | 今天的值日生。"田中"和"佐佐木"。名牌没有被翻到明天那一格 |
| 手机 | 靠窗的课桌上 | 屏幕碎了。但如果你凑近看——最后一条消息是发给妈妈的："今天社团活动取消，我早点回家" |

**叙事逻辑：** 教室是信息密度最高的区域。这里不需要敌人多——需要物件多。玩家在战斗间隙靠到课桌边时，看到桌上的东西。这是故事从"恐怖"转向"悲伤"的转折点。不是"这里发生了什么可怕的事"——是"这里曾经有人的生活，然后戛然而止了"。

#### 体育馆（y <= 700，Boss 间）

| 物件 | 位置 | 暗示什么 |
|------|------|---------|
| 体育器材室的门半开着 | 体育馆右侧 | 里面是整齐码放的篮球、跳绳、体操垫。佐藤生前整理的最后一件事 |
| 点名板 | 体育馆入口墙上 | 三年二班。名单上有些人被划掉了——不是被"什么"划掉的，是普通地用红笔划的。可能是那天请假的同学 |
| 哨子挂绳 | 地板角落 | 一根断了的黑色挂绳，哨子不在上面。哨子在佐藤那里 |
| 跳马箱 | 体育馆左侧 | 翻倒的。五个叠在一起。最上面那个有手掌印——不是血手印，是粉笔灰印。体育课上的正常痕迹 |
| 篮球 | 场地中央 | 一个。停在地板正中央。好像有人在罚球线投了最后一球 |

**叙事逻辑：** 体育馆是佐藤的圣殿。物件的设计目标不是"恐怖"——是"像一个体育老师刚刚离开五分钟"。每一个物件都指向佐藤这个人：他整理器材、他点名、他吹哨。他从来没有离开过这里。

### 2.3 碎片拼图（玩家可能的发现路径）

以下是三条玩家可能经历的"发现路径"——不是强制剧情，而是环境物件自然引导的阅读顺序。

**路径 A：细致型玩家**
1. 操场上看到公告栏日期 → 意识到时间停在某一天
2. 玄关看到鞋柜 → 意识到没有人回家
3. 走廊看到社团海报 → 意识到这些学生有名字、有爱好
4. 教室看到写到一半的笔记 → 意识到一切中断得毫无预兆
5. 体育馆看到点名板 → 意识到佐藤记得每一个学生

**路径 B：直奔 Boss 型玩家**
1. 战斗中忽略大部分物件
2. 在 Boss 战第四乐章看到"你记得怎么敬礼吗" → 产生疑问
3. 战后回到教室探索 → 找到合照、笔记、手机消息
4. 二周目带着认知重新看一切

**路径 C：什么都不看的玩家**
- 也没问题。Boss 战本身就是一个完整的故事——四乐章从"上课"到"下课"，不需要文字也能感觉到。
- 但如果有玩家停下来看了，我们给的每一个细节都能撑住他们的好奇心。

## 三、佐藤幸雄 — 一个人的肖像

### 3.1 他是什么样的人

佐藤幸雄。42 岁。三年二班班主任兼体育教师。

学生们私下叫他"铁人"——不是因为严厉，是因为听说他年轻时单手举起过一台翻倒的摩托车。实际上他从来没举过摩托车。那个传说是他以前教过的一个学生在毕业典礼上编的。佐藤听说后笑了一整个下午。

他是一个会在下雨天把室外体育课改成"体育馆里的躲避球大赛"的老师。他吹哨子的时候，腮帮子会鼓起来，前排的女生会偷笑。他自己也知道。但他还是用力吹。

他从来没有缺席过一堂课。包括最后一堂。

### 3.2 五年前，体育馆

那天上午第三节课，三年二班在体育馆上体育课。热身运动刚做完，佐藤吹哨让大家整队，准备分组打排球。

然后世界变了。

体育馆的门被从外面撞开。不是变异体撞的——是别的班的学生，在逃。佐藤只想了三秒钟。他让学生从后门跑。他站在前门。他吹了集合哨——这是他唯一能想到的事。哨声是体育课上最熟悉的声音。他相信学生会听到。

没有学生来。不是因为他们不听——是因为他们听不到了。或者他们已经不是他们了。

佐藤的身体在那一刻开始变化。但他的意识没有完全消失。他的最后一缕清醒的念头卡在了"集合"这个动作上。从那以后，他一直在吹哨。一直在等。

### 3.3 叙事约束：佐藤不是"怪物"

这个约束影响一切：对话、动作设计、胜利文本、死亡文本。

| 不能说 | 要说 |
|--------|------|
| "Boss 狂暴了" | "他开始冲刺了" |
| "消灭体育老师" | "让体育课结束" |
| "佐藤的攻击" | "他的哨声/他的冲刺/他的投掷"（用具体动作代替"攻击"二字） |
| "他变成了怪物" | "他的身体变了" |
| 任何"邪恶""疯狂""嗜血"等形容词 | 不形容。用动作本身说话 |

佐藤的每一个攻击动作——整队吹哨、分组投掷、长跑冲刺——都是体育课的正常流程。只是扭曲了。这比"怪物攻击你"更有力量：一个你认得的动作，被拉长、被扭曲、被指向你。

### 3.4 佐藤的"最后一件事"

不是锁门。不是保护学生。不是战斗。

是**整理器材室**。

异变发生前的那天早上，佐藤比平时早到了一个小时。因为前一天放学后的篮球部活动把器材室弄乱了。他把篮球码齐、跳绳缠好、体操垫摞成五层。他做完这一切，拍了拍手上的灰，在点名板上确认了今天请假的学生。然后吹哨。上课。

玩家走进体育馆时，器材室的门半开着。里面整整齐齐。这是佐藤留给这个世界的最后一样东西：一间收拾好了的器材室。因为他是一个会在下课后把东西放回原位的人。

这个信息**不需要任何文字告诉玩家**。玩家推开门看到整齐的器材室，可以自己得出结论。如果玩家没推门——也没关系。它在那里。

## 四、主角的目的 — 不只是"活下来"

### 4.1 可视目标：校门外的光

玩家从 CharSelect 按下"推开校门"的那一刻起，屏幕上方——地图的最南端——有一个地方叫"校门"。光照计算（待对齐：需要程序确认校门区域的光照是否可以做特殊处理）让校门外的光线比学校里任何地方都亮。

玩家第一次进入操场时，可以看到校门。但走不过去。有东西挡着。

**这个视觉目标贯穿全程：**
- Stage 1 开场：校门在视野里，但被操场上的敌人隔开
- Stage 2 教室：窗户偶尔能看到操场对面的校门。光是冷的
- Stage 3 体育馆：看不到校门了。但你记得它在哪
- Boss 战胜利后：10% 概率的独白"校门开了。"——四个字，呼应开场

这就是玩家的目的。不是"打败 Boss"——是"看到校门外的光"。

### 4.2 隐藏目的：他是三年二班的学生

文本中只有一处暗示这个信息：Boss 名称揭示时的"三年二班 体育教师 · 佐藤 幸雄"。

为什么是三年二班？因为主角可能就是三年二班的学生。五年前，他从体育馆的后门跑了。他跑了五年。今天他回来了。

这个信息**刻意不确认**。玩家可以：
- 不注意到这个细节 → Boss 战照样成立
- 注意到了 → 一切都不一样了：那些课桌可能是他同学的，那张合照里可能有他，那个哨声他五年前就听过

二周目时这个暗示会变得很重。一周目不需要。

### 4.3 主角不说的话

主角在这整个故事里，**不说的东西比说的东西更重要**：

| 主角不解释 | 为什么不解释 |
|-----------|-------------|
| 他为什么回来 | 他可能自己也不知道。或者他知道但不愿意对自己承认 |
| 他的天赋是什么 | "天赋适应"这个设定太轻小说了。让它留在 CharSelect 里，进了游戏之后再也不提 |
| 他和佐藤的关系 | 不确认。留下空间 |
| 异变是什么 | 一个学生不会用分析的口吻描述世界末日。他只会看到具体的东西：校服、笔记、合照 |

主角的对话策略：他在 Stage 1 说一句，Stage 2 说一句，Boss 战后说一句（或不说话）。一共三句。每一句都是对眼前具体事物的观察，不是对处境的评价。

## 五、三个 Stage 的叙事节奏

### 5.1 Stage 1 — "校门"

**情绪弧线：** 困惑 → 警觉 → 一个让你停下来的发现

**叙事任务：**
1. 建立世界的异常感（不是通过解释，是通过细节）
2. 教会玩家"这里的东西值得看"
3. 给出第一个情感锚点：穿着校服的变异体

**对话草案：**

```
[Stage 1 完成时]

主角："操场上的东西……穿着校服。和我们一样的校服。"
？？："你的心跳变快了。但你没跑。"
```

**设计说明：**
- 主角不说"怪物"——他看到了校服。这是整场游戏里最重要的一个视觉信息：敌人不是外来怪物，是这里的学生。
- "？？"的声音（天赋的回响/残存的意识/什么都可能是）在第一句话就建立了它的身份：它在观察主角的身体反应，语气温和。它不是任务面板。
- "但你没跑"——三个字，确立了主角的性格。不是勇敢，是没跑。区别很重要。

**环境叙事配合：**
- 操场上的敌人（宫崎预放置 6 个）中有 2 个是"站桩型"——玩家不靠近就不动。这让玩家第一次感受到：这个世界里有些东西在等待，而不是在狩猎。
- 公告栏日期在玩家必经之路上。不需要停下。余光可见。

**音乐/音效情绪：** 待对齐（音频系统尚未开发）

**进场方式：** 待对齐（小岛任务流程重构）

---

### 5.2 Stage 2 — "教室"

**情绪弧线：** "这里曾经是活的" → 寻找 → 一个让你移开目光的发现

**叙事任务：**
1. 通过教室物件让玩家建立"这里曾经有人"的认知
2. 抽屉合照是情感锚点——最具体的"曾经的人"
3. 课桌交互（如果实现了）是玩家主动接近物件的机会

**对话草案：**

```
[Stage 2 完成时]

主角："课桌上是上学期没写完的笔记。他们走得很快。"
？？："抽屉里有一张合照。别看了。"
```

**设计说明：**
- "他们走得很快"——不是"他们死了"。主角在用"走"这个字。这是在回避还是在描述？玩家自己判断。
- "？？"的第二句"别看了"——这是整个游戏里最温柔的一句警告。它不是"前方高能"式的提示。它是知道玩家看到了合照、知道那会让玩家难受、但它还是轻轻提醒了一声。玩家可以选择不看。也可以选择看。没有惩罚。

**环境叙事配合：**
- 教室内的课桌（当前已实现 65 个/教室）是很好的叙事载体，但 65 个太多。建议：其中 3-5 张课桌有"可读"内容——不是弹出文本，是用更小的字体在课桌表面显示一行字。玩家靠近就能看到。（待对齐：需要交互策划确认实现方式）
- 讲台上的作业本叠——玩家的移动路径会经过讲台附近。作业本不需要交互，只是一个可见的静态物件。

**可选目标（待对齐：任务系统是否支持可选 Objective）：** "触碰教室里的课桌"——触碰 3 张课桌后完成。不强制。奖励一个 narrative 浮动文字："有些记忆不是你的。但你替他们记住了。"

---

### 5.3 Stage 3 — "体育馆"

**情绪弧线：** 低头往里走 → 看见他 → 理解他 → 让他下课

**叙事任务：**
1. 30 秒接近时间建立"走进别人的领域"的压迫感
2. Boss 战四乐章讲述一堂完整的体育课——从整队到下课
3. 结束时不欢呼。安静。

**对话草案：**

```
[Stage 3 激活时 — 任务提示]

"哨声还在响"

[Boss 名称揭示 — 沉默时刻结束后，屏幕上方]

"三年二班 体育教师 · 佐藤 幸雄"

[四乐章标题 — 每个乐章开始时浮动 1.8s，不冻结]

第一乐章 — 整队
```

**设计说明：**
- 四乐章标题来自对话重写最终版（整队/分组对抗/长跑/下课铃）。它们构成了一堂体育课的自然流程，每个都被扭曲。
- "你记得怎么敬礼吗"——这句话是整个游戏叙事的顶峰。它不问"你记得佐藤老师吗"。它问"你记得怎么敬礼吗"——一个具体的、身体的动作。敬礼是日本学校体育课的开始和结束动作。如果你记得怎么做，说明你上过他的课。如果你上过他的课——你是谁？
- 胜利独白使用概率轮播而不是固定文本。70% 的玩家会看到"灯闪了一下"——一个客观现象。20% 会看到"手在抖"——身体的真实反应。10% 会看到"校门开了"——从开场就在等的四个字。这个概率设计让"校门开了"成为真正的稀有事件——玩家在论坛上会讨论："什么？你有校门开了那句？我怎么没有？"
- 死亡文本"再来一圈"——佐藤不恨你。你倒了，他说再来一圈。像一个体育老师会对跑不动的学生说的那样。这才是最让人难受的死亡文本。

**环境叙事配合：**
- 体育馆的器材室（半开的门，整齐的内部）
- 点名板（挂在入口墙上）
- 篮球在地板中央
- 哨子挂绳在角落

## 六、Boss 战叙事高潮设计

### 6.1 登场序列

| 时间点 | 事件 | 叙事效果 |
|--------|------|---------|
| t=-30s | Stage 3 激活，箭头指向体育馆深处 | 你不确定里面是什么。但你知道必须过去 |
| t=-30s ~ t=-3s | 暗角渐变，普通敌人持续刷新 | 你还在战斗。但空气变了 |
| t=-3s ~ t=0s | HUD 阶梯消退，沉默时刻 | 你看见角落里巨大的阴影。它一直在那里 |
| t=0s | Boss 光环首次亮起，名称浮出 | "三年二班 体育教师 · 佐藤 幸雄"——十六个字，屏幕上方 |

### 6.2 四乐章的情感曲线

```
第一乐章 "整队" — 情感：不解
  你不知道这个红色巨块是什么。它吹哨。你躲。
  叙事任务：建立"这曾经是一个人"的前提。

第二乐章 "分组对抗" — 情感：应对
  球飞出来。不是篮球。你开始理解它的模式。
  叙事任务：让玩家意识到"这些动作我见过"。

第三乐章 "长跑" — 情感：恐惧与悲哀交替
  跑道延伸。白影学生跑进来。它第三次吹哨——
  没有人来。那 0.8 秒的停顿。是整个战斗最响的声音。
  叙事任务：空哨。让"没有学生了"成为一个事件而不是一个状态。

第四乐章 "下课铃" — 情感：复杂
  没有光环了。它在冲刺。你也在冲刺。
  "你记得怎么敬礼吗。" —— 一个问题。你要用最后一击来回答。
  叙事任务：战斗结束在问题里。答案不是"是"或"否"——答案是你做了什么。
```

### 6.3 "空哨"时刻 — 叙事设计的核心

第三乐章中，佐藤第三次吹哨召唤学生。没有学生来。

这个时刻的设计细节：

1. **视觉：** 白色声波从 Boss 位置扩散——和前两次一样
2. **0.5s 后：** 声波消散。没有白影出现
3. **Boss 停住：** 0.8s 静止。move_speed = 0。不攻击
4. **然后：** Boss 自己冲上来。第四乐章的序幕

这 0.8 秒是什么？是玩家第一次真正看见佐藤——不是作为 Boss，是作为一个等了五年的人。他吹了哨，没人来。他已经知道不会有人来了。但他还是吹了。

这 0.8 秒里不应该有任何文字。如果一定要有，只有三个点："..."

### 6.4 终结序列

| 时间点 | 事件 | 叙事效果 |
|--------|------|---------|
| 最后一击 | Big Hit Stop (0.1s) | 世界停了一帧 |
| Hit Stop 后 | Boss 坍塌。不是爆炸——是倒下。横向拉宽，垂直压缩 | 一个重量级的结束 |
| 坍塌中 | 学生残影全部静默消散 | 不是死亡。是下课。他们可以走了 |
| 坍塌完成 | 金色粒子从倒地位置爆发 | 金色 = 自由 |
| 粒子中 | "佐藤 幸雄 —— 下课。" 浮现 | 句号。不是欢呼 |
| 2s 后 | 胜利独白（概率轮播） | 给这个时刻一个轻轻的收尾 |

## 七、两个结局

### 7.1 胜利结局

```
最后一击落下。
他倒下的时候，体育馆的灯闪了一下。你站在原地。手在抖，或者没有。
校门开了。光从那边进来。
他没有被"消灭"。他下课了。
```

**胜利后的设计考虑：**
- 没有庆祝动画。没有升级面板立刻弹出来（延迟到 2s 后）
- 如果玩家因此升级，面板标题是"他留给你的"——五个字，足够
- 如果触发 Operator Protocol，"哨声"选项的文字是"你听见了哨声。你应该听不见的。"——不是奖励，是困惑。是佐藤的一部分进入了你的天赋里。这是馈赠还是负担？你自己判断
- 校门外的光应该比学校里任何区域都亮。玩家走出去。游戏结束——或者继续，如果有后续内容的话

### 7.2 失败结局

```
你倒下了。
视线模糊的时候，哨声还在响。
"再来一圈。"
```

**失败后的设计考虑：**
- Game Over 面板的色调改为灰白 `Color(0.7, 0.7, 0.7)`，不是默认白色
- 背景暗角加深
- 没有"你死了"或"你阵亡了"——佐藤说了"再来一圈"
- 如果没有设专用的 Boss 战 Game Over 文本，回退到默认 Game Over。但必须设。这是 Boss 战完整性的最后一块拼图

## 八、叙事设计检查清单（自检）

以下问题来自陶德的审核标准。每个答案必须在实现前确认。

### 玩家自由

- [ ] 玩家能不能跳过所有对话，纯靠视觉理解故事？
  - **能。** Boss 的四乐章动作设计本身就是一堂体育课的叙事。环境物件不需要交互——它们在那里，玩家可以选择看或不看。
- [ ] 玩家能不能忽略主线，在学校里做自己的事情？
  - **当前限制：** Stage 系统线性的。Stage 1 → Stage 2 → Stage 3。但玩家在每个 Stage 内可以自由移动、打怪、探索。**待对齐：** 是否允许玩家从 Stage 1 直接走到 Stage 3 触发区域？如果宫崎的地图允许路径自由，那就让玩家自由——只是难度很高。
- [ ] 如果玩家想到了一个奇怪的玩法，系统能不能支持？
  - **故事层面：** 可以。故事不是"剧情触发"式的——是环境物件 + Boss 战动作。玩家用什么方式击败佐藤，不影响故事成立。

### 世界沉浸

- [ ] NPC 在没有玩家的时候在做什么？
  - **待对齐：** 当前敌人 AI 没有"闲置行为"——所有敌人要么追踪玩家要么站桩。如果技术允许，给站桩型敌人加一个微小的闲置动画（如原地微微晃动），暗示"它们也在等"。
- [ ] 这个世界的规则对玩家和对 NPC 是不是一视同仁？
  - **是。** 佐藤用的"技能"（哨声、投掷、冲刺）遵循和玩家技能相同的 CombatUnit 框架。他不是"Boss 特权"——他是另一个 CombatUnit 上挂着不同的配置。

### 水冷器时刻潜力

- [ ] "空哨"——玩家会不会停下来那 0.8 秒？
- [ ] "你记得怎么敬礼吗"——玩家会不会在屏幕前愣了一下？
- [ ] "校门开了"——10% 概率的稀有独白。看到的人会不会去 Reddit 发帖？
- [ ] 抽屉里的合照——发现它的玩家会不会在 Discord 里截图分享？
- [ ] 器材室整整齐齐——谁会注意到？注意到的人会怎么跟没注意到的人描述？

## 十一、最终的话

这份故事设计建立在一个信念上：**最好的学校恐怖故事，不是"有人在追你"——是"这里曾经有人，但他们不在了，而你还记得他们。** 

佐藤不是一个等着被杀死的 Boss。他是一个等着下课的老师。那个哨子在他喉咙里卡了五年。你走进体育馆的那一刻，不是去战斗——是去让他把哨子放下来。

这不是一个关于打败怪物的故事。这是一个关于听到哨声之后走进去的故事。仅此而已。

校门外的光还在。你可以出去了。

## 附录 B：Narrative Bible（叙事准则，后续副本复用）

1. **不要解释异变。** 它不是故事。它是故事的背景。越解释越没意思。
2. **每个 Boss 是一个人。不是怪物。** 有名字、有身份、有最后一件事。玩家不是在杀 Boss——是在帮他们结束最后一件事。
3. **主角是观察者，不是英雄。** 他不宣言。他看到东西，说出来。不说出来的更多。
4. **物件比文字有力。** 一张写到一半的笔记胜过三段背景介绍。一扇关着的门比一句"前方危险"更让人紧张。
5. **留白。** 玩家不需要知道全部真相。他们需要的是足够多的碎片——多到可以拼出属于自己的版本。
6. **翻译成"玩家做了什么"。** 故事不是说出来的。故事是玩家穿过操场、推开教室门、在抽屉里看到合照、然后站在体育馆门口——这整个过程。
