# Boss 数值强化方案 v1.0

> 起草: 宫崎英高 (战斗策划)
> 日期: 2026-05-19
> 状态: 待数值策划确认伤害数值

---

## 1. 问题诊断

反馈：Boss 太弱，弹幕不够密不够广，没有魂系感觉。

根因分析：
- 扇形弹幕仅 3 发，覆盖面小，玩家轻松走位躲避
- 攻击间隔过长（P1 平均 3s），Boss 大部分时间在走路而不是输出
- 无预判瞄准，弹幕指向玩家当前位置，稍微移动即可全部躲开
- 前摇可读性不足，玩家看不出 Boss 什么时候出招
- 伤害偏低，即使中弹也没有压迫感

---

## 2. 修改方案

### 2.1 弹幕密度翻倍（阶段递进）

| 乐章 | 弹丸数量 | 扇形角度 | 弹丸间距 | 视觉感受 |
|------|---------|---------|---------|---------|
| P1 热身 | **5** | 90 deg | 22.5 deg | 基础压力 |
| P2 球类训练 | **7** | 100 deg | 16.7 deg | 弹幕开始密集 |
| P3 集合 | **9** | 110 deg | 13.8 deg | 很难全身而退 |
| P4 毕业考试 | **11** | 120 deg | 12.0 deg | 弹幕地狱 |

对比原版：全阶段固定 3 发 / 90 deg。P4 密度提升 3.7 倍。

### 2.2 弹幕速度递增

| 乐章 | 弹丸速度 (px/s) | 说明 |
|------|----------------|------|
| P1 | 200 | 慢速，玩家可反应 |
| P2 | 250 | 中速 |
| P3 | 300 | 快速 |
| P4 | 350 | 极速，需要预读走位 |

原版全阶段统一 180 px/s。

### 2.3 攻击间隔递减

| 乐章 | 攻击间隔 | 说明 |
|------|---------|------|
| P1 | **3.0s** | 从容，给玩家熟悉节奏 |
| P2 | **2.2s** | 节奏加快 |
| P3 | **1.5s** | 弹幕连续不断 |
| P4 | **1.0s** | 几乎无缝弹幕 |

原版：P1 2.5-3.5s, P2 2.0-3.0s, P3 1.5-2.2s, P4 0.8-1.5s。区间过大导致体感不一致，改为固定值确保节奏稳定。

### 2.4 预判弹幕

新增 `_get_predictive_direction()` 算法：

```
预测位置 = 玩家当前位置 + 玩家速度 * 飞行时间
飞行时间 = clamp(距离 / 弹丸速度, 0.15s, 0.5s)
```

效果：弹幕飞向玩家移动方向的前方，而非当前站位。
- 站桩玩家：预判 = 当前位，无影响
- 横向移动玩家：弹幕打前方，命中率显著提升
- 变速走位玩家：需要预测 Boss 的预测，产生魂系博弈感

### 2.5 魂系可读性强化

**前摇脉冲动画** (`BossAI._flash_boss_windup`)：
- 前摇期间 Boss 体色周期性闪红白 (Color(1.3, 0.3, 0.2))
- 脉冲次数 = ceil(duration / 0.2)，最少 2 次
- 前摇结束瞬间清除视觉

**攻击帧闪光** (`WhistleWave._show_shoot_flash`)：
- 弹幕发射瞬间 Boss 亮白 0.06s，然后渐变回本色
- 类似魂系列 Boss 举刀的最高点闪光

**硬直窗口**：
- 扇形弹幕恢复时间从 0.4s 提升至 **0.5s**
- 每一轮弹幕释放后玩家有 0.5s 安全输出窗口
- 此窗口内 Boss 不可攻击、不可移动

### 2.6 伤害数值调整

| 乐章 | 接触伤害 | 弹丸伤害 | 备注 |
|------|---------|---------|------|
| P1 | 22 | 15 + 0*4 = **15** | 热身 |
| P2 | 25 | 15 + 1*4 = **19** | 球类训练 |
| P3 | 28 | 15 + 2*4 = **23** | 集合 |
| P4 | 35 | 15 + 3*4 = **27** | 毕业考试 |

弹丸伤害公式：`base_damage (15) + phase_index * 4`

防御力同步递增：P1=5, P2=8, P3=12, P4=15

### 2.7 技能槽位重新分配

| 乐章 | 旧 skill_slots | 新 skill_slots | 变更说明 |
|------|---------------|---------------|---------|
| P1 | [1, 2] | [1, 2, **4**] | +RollCharge 增加近战压力 |
| P2 | [3, 4] | [**1**, 3, 4] | +WhistleWave(7发)为弹幕核心 |
| P3 | [0, 5] | [0, **2**, 5] | +WhistleWave(9发)，召唤间隔从10s缩短到8s |
| P4 | [7, 6] | [**1**, 6, 7] | +WhistleWave(11发) + 全屏器材雨 + 绝望冲刺 |

核心设计理念：**每一乐章都以扇形弹幕为输出主干**，辅以其他技能制造变化。

## 5. 测试要点

1. **P1 热身**：5发弹幕间隔 3s。站桩应该中 1-2 发。横向慢走应该全躲。
2. **P2 球类训练**：7发弹幕 + 直球补充。中距离应该感受到压迫。
3. **P3 集合**：9发弹幕 + 学生围堵 + 冲刺。召唤间隔 8s 是否合理。
4. **P4 毕业考试**：11发弹幕 1s 间隔 + 器材雨 + 绝望冲刺。弹幕地狱感。
5. **预判弹幕**：持续横向移动时弹幕明显打在前方。变速走位能有效规避。
6. **前摇可读**：Boss 身体闪红白时马上移动，可以躲开。
7. **硬直窗口**：攻击结束后约 0.5s Boss 不动，可以打一套。
8. **阶段切换**：HP 75%/50%/25% 触发，后撤 + 闪烁 + 光环变色。
