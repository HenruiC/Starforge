# 副本蓝图编辑器 — EventGraph 调研

**结论：适合 fork。**

1. **协议**：MIT，可任意修改和商用，完全兼容。
2. **架构**：EditorPlugin 注册 → `add_control_to_bottom_panel()` 嵌入 GraphEdit 编辑器；节点继承 `EventNodeResource`，由 `EventNodeRegistry` 自动发现注册，扩展 API 极简。
3. **五类节点工作量**：很低。已有 Event/Flow/Action/Logic/Data 五类等价内置节点，新增仅需编写 `EventNodeResource` 子类（每类约 20-50 行 GDScript）放入对应目录，零侵入。估计总工作量 1-2 天。

**风险**：v1.0.0，文档偏少，需实测 GraphEdit 在 Godot 4.4 的稳定性。
